# JRE 
JRE 是 Java 运行环境，主要用于运行 Java 程序。它包含：
- JVM（Java Virtual Machine，Java 虚拟机）
- Java 核心类库
- 运行 Java 程序所需的支持文件
如果你只是想运行 Java 程序（比如运行一个 .jar 文件），安装 JRE 就够了。

# JDK
JDK 是 Java 开发工具包，用于开发和运行 Java 程序。 它包含：
- JRE（所以也包含 JVM 和核心类库）
- Java 编译器（javac）
- 调试工具（jdb）
- 文档生成工具（javadoc）
- 打包工具（jar）等开发工具
如果你要编写、编译 Java 程序，必须安装 JDK。


JDK = JRE + 开发工具

JRE = JVM + 核心类库
